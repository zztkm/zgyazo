# 仕様

Go で以下の仕様を満たすアプリケーションを実装する。

- このアプリは Windows でバックグラウンド常駐型のアプリケーションである
- スニッピングツールでキャプチャした画像が保存されるディレクトリを監視する
    - このディレクトリは Snipping Tool で設定されたディレクトリで、このアプリの config には同じディレクトリを指定する
- 監視するディレクトリに画像が保存されると、以下の処理を行う
    - 画像を Gyazo にアップロードする
    - Upload API のレスポンスに `parmalink_url` が含まれるので、これを取得する
    - 取得した `parmalink_url` をブラウザで開く
    - config でアップロード成功後に削除するように設定されている場合、監視するディレクトリから画像を削除する

その他検討事項

- `Ctrl + Shift + C` でスニッピングツールを起動するように設定する
  - これは外部のツールを使ってもらうか、アプリケーション内で実装するかは検討中
